# Zhipu Tray 设计文档

> **日期:** 2026-04-09
> **目标:** 为 Zhipu ASR 语音输入法创建桌面托盘应用

## 概述

基于 Arc Linux KDE Plasma 桌面，将 asr_cli.py 封装为带有系统托盘的 GUI 应用。

## 系统架构

```
┌─────────────────────────────────────┐
│  zhipu_tray.py (入口)               │
│  - 创建系统托盘                       │
│  - 显示主窗口                        │
└──────────────┬──────────────────────┘
               │
       ┌───────┴───────┐
       ▼               ▼
┌─────────────┐  ┌─────────────┐
│ TrayIcon    │  │ MainWindow  │
│ - 静态图标   │  │ - 状态显示   │
│ - 动画状态   │  │ - 设置面板   │
│ - 右键菜单   │  │ - 日志查看   │
└─────────────┘  └─────────────┘
       │               │
       └───────┬───────┘
               ▼
┌─────────────────────────────────────┐
│  ASREngine (封装 asr_cli.py)         │
│  - 录音控制                         │
│  - ASR 识别                         │
│  - 自动打字                         │
└─────────────────────────────────────┘
```

## 技术栈

- **GUI 框架:** PySide6 (Qt 官方 LGPL 许可)
- **系统托盘:** Qt SystemTrayIcon
- **核心引擎:** 复用 asr_cli.py 的 ASR 逻辑
- **Python:** 3.13

## 托盘交互

| 状态 | 图标 | 说明 |
|---|---|---|
| 监听中 | 静态灰麦克风 | 等待右 Ctrl |
| 录音中 | 脉冲/呼吸动画 | 按住右 Ctrl 时 |
| 识别中 | 旋转动画 | 松开右 Ctrl，正在识别 |

## 托盘右键菜单

- 设置 API Key
- ───────────
- 退出
